# CanvasForge — Kullanıcı Acı Noktaları

## Jira Kullanıcıları

1. **Karmaşık yapılandırma:** İlk kurulum 2–3 gün sürüyor
2. **Yavaş arayüz:** Sayfa yüklemeleri 3–5 saniye
3. **Pahalı lisans:** 10 kişilik ekip için ~$900/yıl
4. **Manuel sprint planlama:** Velocity hesabı elle yapılıyor
5. **Türkçe kalitesi:** Çeviri eksik ve tutarsız

## Trello Kullanıcıları

1. **Sprint desteği yok:** Ayrı Power-Up gerekiyor
2. **Bağımlılık yönetimi yok:** Kartlar arası ilişki kurulamıyor
3. **Raporlama zayıf:** Burndown, velocity grafikleri yok
4. **Kurumsal özellikler eksik:** Audit log, SSO yok

## Türk Yazılım Ekipleri (Ortak)

1. **KVKK uyumu:** Veri işleme kayıtları manuel tutuluyor
2. **Resmi tatil takvimi:** Sprint tarihleri tatillerle çakışıyor
3. **Yerel destek:** İngilizce-only destek kanalları
4. **Fiyat hassasiyeti:** Dolar bazlı fiyatlandırma TL dalgalanmasından etkileniyor

## CanvasForge Çözümleri

| Acı Noktası | Çözüm |
|-------------|-------|
| Karmaşık kurulum | 5 dakikada workspace + ilk tahta |
| Manuel sprint planlama | Velocity-aware otomatik öneri |
| Bağımlılık kör noktası | Yerleşik dependency graph |
| KVKK belirsizliği | Tam denetim izi + veri işleme kaydı |
| Tatil çakışması | Türk resmi tatil takvimi entegrasyonu |
